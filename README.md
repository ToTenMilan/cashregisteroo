### 1. Clone the repo

```sh
git clone https://github.com/ToTenMilan/cashregisteroo.git
```

### 2. Test

##### Automatic

run tests by
```sh
ruby test_cash_register.rb && ruby test_product.rb && ruby test_receipt.rb
```

##### Manual

Run main file
```sh
ruby main.rb
```

##### Dependencies

Tested on:
ruby 3.1.4p223 (2023-03-30 revision 957bb7cb81) [x86_64-darwin22]
